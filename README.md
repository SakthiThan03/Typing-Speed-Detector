# Typing Speed Detector

A web-based typing speed application built with TypeScript, HTML, CSS, and Vite. The application measures typing speed (Words Per Minute), typing accuracy, and completion time while providing a multi-screen user experience.

## Features

* Welcome screen with test launch button
* Random quote generation
* Real-time WPM calculation
* Real-time accuracy tracking
* Live timer
* Results screen displaying final performance metrics
* Restart and retry functionality
* Modular TypeScript architecture

## Technologies Used

* TypeScript
* HTML5
* CSS3
* Vite
* Git
* GitHub

## Installation

```bash
git clone https://github.com/SakthiThan03/Typing-Speed-Detector.git
cd Typing-Speed-Detector
npm install
npm run dev
```
